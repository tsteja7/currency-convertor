# INR to USD Currency Converter

A simple web page that converts an amount entered in Indian Rupees (INR) into United States Dollars (USD).

## Files

- `index.html` - page markup
- `styles.css` - styling for the converter page
- `script.js` - conversion logic and event handling

## Usage

1. Open `index.html` in a web browser.
2. Enter an amount in the INR field.
3. The USD value updates automatically.

## Notes

- The converter uses a fixed exchange rate of `0.012` USD per INR.
- Adjust `INR_TO_USD_RATE` in `script.js` for a different conversion rate.
